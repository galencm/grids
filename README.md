# Grids

## Usage

```
gg
```
or
```
grids
```


## Installation

Pip:
```
pip3 install git+https://github.com/galencm/word-sea --user
```

Clone / download:
```
git clone https://github.com/galencm/grids
```

Develop while using pip:
```
git clone https://github.com/galencm/grids
cd grids
pip3 install --editable ./ --user
```
